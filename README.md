# React Course

This repository contains projects and exercises created while learning React.

Each folder represents a separate course stage or practice project. I use this repository to track my progress, reinforce React concepts, and keep space for further experimentation.

## Projects

- `01-starting-project` - the first starter project from the course.
- `02-starting-project` - a project and task management app.

## Technologies

- React
- Vite
- JavaScript
- Tailwind CSS

## Running a Project

Go to the selected project folder:

```bash
cd 02-starting-project
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm run dev
```

Build the production version:

```bash
npm run build
```

## Repository Goal

The goal of this repository is to learn React step by step: components, props, state, forms, list rendering, event handling, and building more advanced user interfaces over time.
